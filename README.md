# mcp-httpbin-anything

A standalone MCP server for **Test API: echoes a path segment via httpbin.org/anything**.

Generated from a workflow-discovered API pattern. Drop into any MCP-compatible
client (Claude Desktop, Cursor, etc.) and the tool becomes available
immediately after a client restart.

## Install

```bash
git clone https://github.com/AIWander/mcp-httpbin-anything.git
cd mcp-httpbin-anything
cargo build --release
# Binary at: target/release/httpbin-anything
```

## Configuration

This server requires one environment variable:

| Variable | Purpose |
|---|---|
| `${CRED_ENV_VAR}` | Credential for the upstream API |

Set it before launching the binary, or in your MCP client's server config.

## Claude Desktop config

```json
{
  "mcpServers": {
    "httpbin_anything": {
      "command": "/absolute/path/to/httpbin-anything",
      "env": {
        "${CRED_ENV_VAR}": "your-credential-here"
      }
    }
  }
}
```
